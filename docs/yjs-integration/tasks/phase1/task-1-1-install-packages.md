# Task 1-1: パッケージインストール

## 概要

Yjsライブラリと関連パッケージをインストールする

## 対象ファイル

- `package.json`

## 依存タスク

なし（最初のタスク）

## 実装内容

以下のコマンドを実行してパッケージをインストール:

```bash
pnpm add yjs y-protocols
```

### パッケージ説明

| パッケージ | 説明 |
|-----------|------|
| `yjs` | CRDTベースのリアルタイム同期ライブラリ本体 |
| `y-protocols` | Yjsのネットワークプロトコル（sync, awareness等） |

## 完了条件

- [ ] `yjs`がインストールされている
- [ ] `y-protocols`がインストールされている
- [ ] `package.json`に依存関係が追加されている
- [ ] `pnpm-lock.yaml`が更新されている
